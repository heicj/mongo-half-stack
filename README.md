# How To Use
## GET requests to /dogs
* `/dogs` 
    * returns **all dogs in db**

* `dogs/id`
    * returns **dog with that id**

## POST request to /dogs
* adds dog to db

## PUT request to /dogs/id
* updates **dog with that id**

## DELETE request to /dogs/id
* deletes **dog with that id**
